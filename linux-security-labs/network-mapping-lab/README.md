# Network Mapping Lab

## Overview

This lab will document basic network mapping with Nmap. The goal is to practice scanning a local lab network, identifying hosts and services, and interpreting scan results responsibly.

## Objectives

- Identify a local lab network range
- Run basic Nmap host discovery
- Scan selected hosts for open ports
- Interpret Nmap output at a beginner level
- Document findings without exposing sensitive information

## Primary Tool

- `nmap`

## Planned Workflow

1. Confirm the local lab network range.
2. Run a basic discovery scan.
3. Select a known lab host for deeper scanning.
4. Review detected ports and services.
5. Document results and limitations.
