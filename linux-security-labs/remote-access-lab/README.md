# Remote Access Lab

## Overview

This lab will document basic remote access setup on Linux using SSH. The goal is to practice secure connection setup, key management, and simple service verification in a controlled lab environment.

## Objectives

- Generate SSH key pairs with `ssh-keygen`
- Configure SSH client access
- Review basic `sshd` server configuration
- Manage `authorized_keys`
- Verify remote login behavior
- Document safe remote access practices

## Tools

- `ssh`
- `ssh-keygen`
- `sshd`
- `systemctl`
- `nano`
- `cat`

## Planned Workflow

1. Generate a key pair for lab use.
2. Review public and private key roles.
3. Add the public key to `authorized_keys`.
4. Review basic `sshd` settings.
5. Test SSH access in a controlled environment.
6. Record observations and issues.
